# Code of Conduct

## Our Pledge

We want Bugbird NestGate to be a respectful, practical, and safe project for contributors of all experience levels.

## Our Standards

Examples of behaviour that helps this project:
- being respectful and constructive
- assuming good intent
- reporting security concerns privately
- focusing critiques on the code, docs, or design

Examples of unacceptable behaviour:
- harassment or discrimination
- doxxing
- hostile or abusive language
- publishing exploit details before a fix is available

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that break this code of conduct.

## Reporting

For general conduct issues, open a private conversation with the maintainers.

For security issues, follow the process in `SECURITY.md`.
