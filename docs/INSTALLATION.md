# Installation Guide

## Door Installation

1. Mount controller enclosure on secure side
2. Run strike wiring separately from signal wiring
3. Install flyback diode across strike
4. Confirm correct voltage at strike
5. Test manual unlock pulse

---

## Server Installation

1. Install Raspberry Pi OS Lite
2. Secure SSH access
3. Install MQTT broker with TLS
4. Install NestGate server
5. Create admin account
6. Add doors and roles

---

## Provisioning a Door

1. Flash ESP32 firmware
2. Power on device
3. Retrieve one time claim code
4. Claim device in dashboard
5. Assign door and permissions
6. Verify heartbeat
