# Bill of Materials (Per Door)

## Required

- ESP32 dev board
- Electric strike (12V DC)
- 12V DC power supply
- Buck converter (12V -> 5V)
- PN532 NFC reader
- Logic level N channel MOSFET
- Flyback diode
- Metal enclosure
- Terminal blocks
- Wiring and ferrules

## Recommended

- Door contact reed switch
- Tamper switch (normally closed)
- Status LED
- Buzzer
- Secure element ATECC608A

---

## Server Hardware

- Raspberry Pi 4 or 5
- SSD or high endurance SD card
- Ethernet connection
- Backup storage
- Optional UPS
